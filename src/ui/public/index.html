PPT Workflow Studio

银行金融风格 PPT 工作台

上传需求文档、空白模板、素材 PPT 和参考 PPT / 参考图片。系统会先解析标题结构，再结合参考风格生成可编辑草稿与最终 PPT。

输入支持 DOCX / PDF / 模板 PPT / 素材 PPT / 参考 PPT / 参考图片
核心能力 标题树分页、参考风格拟合、真实渲染预览、JSON 校正

1. 上传材料

参考 PPT 会在后台自动拆成参考页图片，用于学习布局、图标、表格和文字节奏。

语义模型设置

支持本地 OpenAI 兼容服务，也支持 Minimax / OpenAI API。使用 Minimax 时只需要填写 API Key。

2. 处理状态

展示上传、解析、素材融合、动态规划和渲染状态，并实时显示进度条。

3. 素材库概览

按品牌、图标、插画、截图等类别展示缩略图，支持横向滚动预览。

4. 处理流程

从文档解析到草稿渲染，每一步的中间状态都会显示在这里。

5. 文档摘要

识别正文、表格、图片和标题层级，用于后续分页和语义规划。

6. 草稿下载

草稿 PPT、Outline、Style、布局和备注都可以直接下载。

7. 草稿预览

这里展示 PowerPoint 实际渲染截图，用于检查文字重叠、留白和图表位置。

8. 页面级布局模板

每一页都会按页面类型匹配模板，也可以单独切换，用于拉开不同页面的风格差异。

9. JSON 校正

支持直接修改 Outline / Style JSON，并回写到当前草稿后继续生成。
